# RCT Outcome Prediction

A machine learning project for predicting outcomes of Randomized Controlled Trials (RCTs).

## Project Structure

```
RCT-outcome-prediction/
├── data/           # Raw and processed data
├── notebooks/      # Jupyter notebooks
├── src/            # Source code
└── requirements.txt
```

## Setup

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
